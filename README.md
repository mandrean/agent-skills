# agent-skills

Shared repository for agent-specific skills and supporting assets.

## Layout

- `claude/skills/`: Claude-oriented skills and related resources
- `codex/skills/`: Codex skills in the standard `SKILL.md` folder format

## Conventions

- Keep one skill per subdirectory.
- Keep reusable references, scripts, and assets inside each skill directory.
- Document tool-specific setup close to the skill that needs it.
